# Enterprise IT Project Management Artifacts

Welcome to my Project Management methodology repository. 

As a Senior Project Manager leading complex, multi-million-dollar technology initiatives, I rely on highly structured, repeatable processes to ensure delivery excellence. I created this repository to open-source the sanitized templates, governance logs, and workflows I use daily to manage cross-functional teams, control budgets, and maintain stakeholder transparency.

## 📂 Repository Contents
* **Governance & Compliance:** Architecture Review Board (ARB) templates, Change Approval Board (CAB) workflows, and Risk Registers.
* **Agile & Scrum Execution:** Sprint ceremony checklists, backlog refinement frameworks, and User Acceptance Testing (UAT) documentation.
* **Executive Reporting:** Standardized burndown reports and steering committee status updates.
* **AI & Automation:** My personal library of AI prompts (Copilot) used to accelerate meeting documentation, requirement extraction, and reporting.

*Note: All artifacts have been strictly sanitized to remove any proprietary or confidential information.*
